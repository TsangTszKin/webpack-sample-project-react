## Quick Start
#### Down
~~~
git clone https://github.com/TsangTszKin/webpack-sample-project-react.git
~~~

#### Install
~~~
npm install
~~~

#### Run（本地服务器）
~~~
npm run server
~~~

#### Build
~~~
npm run build
~~~

#### 发布CDN(推荐)

将project.config.js内publicPath改为服务器绝对路径。

#### 发布静态资源

若没有CDN，只能发布静态资源，需修改webpack配置，将所有文件放置在同一目录下。
